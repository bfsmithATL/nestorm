## Description

A simple demo REST API project including wiring for TypeORM to MSSQL and a simple configuration service, based on NestJS framework.

## Installation

```bash
$ npm install
```

## Running the app

```bash
To start the API in VSCode debug + watch mode: press F5

# development
$ npm run start

# watch mode
$ npm run start:dev

# debug mode
$ npm run start:debug

# production mode
$ npm run start:prod
```
